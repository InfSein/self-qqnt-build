# self-qqnt-build
自用 QQNT &amp; LiteLoader 构筑合集

## 为什么是 9.9.9 ?
* 功能已经足够使用
* 最后一个支持全窗口磨砂的版本
* 不会出现 LiteLoaderQQNT/LiteLoaderQQNT#1032 问题
